【水的沸腾】：准备一个烧杯，加入适量的水。将烧杯放在酒精灯或电热板上加热。观察水的温度变化，并用温度计记录数据。观察水开始冒泡的现象，记录水沸腾的温度（100°C）。

【胆矾的研碎】：取适量胆矾（硫酸铜晶体）放入研钵中。使用研杵轻轻按压和研磨胆矾，直到其变成粉末状态为止。观察研碎后的胆矾外观，记录其颜色和质地的变化。

【往硫酸铜溶液中滴加氢氧化钠溶液】：准备一个试管，加入适量的硫酸铜溶液（蓝色）。使用滴管，慢慢向试管中滴加氢氧化钠溶液。观察反应现象，记录氢氧化钠溶液加入后的颜色变化（出现蓝色沉淀）。

【石灰石与盐酸的反应】：将适量的石灰石（碳酸钙）放入一个烧杯或试管中。向烧杯中滴加适量的盐酸（稀盐酸）。观察反应，记录气泡产生的情况（二氧化碳气体的生成）。可以用气体产生的泡沫去检验不再产生泡沫时的反应完成度。

【氧气和二氧化碳的有关性质】：准备一个装有氧气的试管和一个装有二氧化碳的试管。取一根易燃物（如火柴或蜡烛），将其点燃。将点燃的易燃物放入装有氧气的试管中，观察燃烧现象（燃烧加剧）。将点燃的易燃物放入装有二氧化碳的试管中，观察现象（火焰熄灭）。记录观察到的氧气和二氧化碳的性质以及它们对燃烧的影响。

【固体药品的取用】 : 首先，准备好所需的固体药品和药匙，确保药品在清洁的实验环境中使用。然后，打开药品的容器，用干净的药匙轻轻搅拌药品，确保其均匀。接着，用药匙小心地取出适量药品，观察药品的性质（如颜色、颗粒大小等），最后将取用的药品放入干净的容器中，避免交叉污染，并将未用完的药品按照实验室规定妥善保存。

【在澄清石灰水中滴加酚酞溶液】 : 先准备一个装有澄清石灰水的试管，然后用滴管吸取适量的酚酞溶液。接着，将酚酞溶液缓慢滴加到石灰水中，观察溶液颜色的变化。当酚酞溶液与石灰水反应后，如果石灰水变为桃红色，说明其为碱性溶液。

【在大理石或碳酸钠粉末中加入稀盐酸】 : 取适量的大理石或碳酸钠粉末放入一个试管中，随后用量筒或滴管吸取适量稀盐酸。将稀盐酸缓慢倒入试管中，观察反应现象，注意观察是否有气泡产生，记录反应的剧烈程度及气泡数量，反应完毕后，确保安全处理试管中的物质。

【观察酒精灯火焰；将火柴梗平放如火焰中】 : 首先，准备一个酒精灯，打开灯芯并点燃，观察火焰的形状和颜色。接着，取一根火柴梗，将其平放在火焰中，观察火柴梗的燃烧情况，记录火焰的状态以及火柴梗燃烧的效果，如火焰的温度和颜色变化。

【在氢氧化钠溶液中滴入硫酸铜溶液后加热】 : 首先，在烧杯中加入适量氢氧化钠溶液。然后，使用滴管吸取适量的硫酸铜溶液，缓慢滴入氢氧化钠溶液中，观察反应并记录反应的颜色变化。接着，将烧杯放置在电热板上加热，观察加热后的反应现象，记录任何沉淀的生成情况。

【连接装置并检查装置的气密性】 : 准备所有所需的装置和连接管。按照实验要求将各个部件连接起来，确保连接紧密。检查连接处是否有漏气现象，可以用湿润的手指触摸连接点，如有气泡产生表示有漏气。确保设备的气密性后，可以进行后续的实验步骤。

【洗涤玻璃仪器】 : 首先，准备需清洗的玻璃仪器（如试管、烧杯等），并在水槽中放入清水。然后，使用洗涤剂和刷子对玻璃仪器进行清洗，确保去除残留物。清洗完毕后，用干净的自来水彻底冲洗设备，直至没有洗涤剂的残留，最后倒置玻璃器具让其自然晾干。

【测定空气中氧气的含量】 : 首先，准备一个装有适量氧化镁的试管，并在试管上方放置一个气体集气瓶。然后，用加热器加热试管中的氧化镁，直到其完全反应，形成氧化物。通过计算反应前后试管中气体的体积变化，来测定空气中氧气的含量，注意在整个过程中要防止气体的泄漏以保证结果的准确性。

【带火星的木条复燃】 : 取一根干燥的木条，用火柴点燃并让其燃烧几秒钟，之后迅速将其放入一个装满氧气的瓶中。观察木条的状态，注意是否会复燃，并记录复燃的现象及时间，比较在开放空气与氧气环境中木条的燃烧情况。

【硫分别在空气和氧气里燃烧】 : 首先，准备两支试管，一支装有空气，另一支装有纯氧气。取适量硫磺，点燃后迅速放入试管中。观察并记录硫在空气中的燃烧情况以及在氧气中的燃烧现象，比较两者的火焰颜色、光亮程度及燃烧速度。

【细铁丝在空气中红热，在氧气中剧烈燃烧】 : 取一根细铁丝，将其一端固定在夹具上，然后用酒精灯或其他加热源加热铁丝，使其变红。当铁丝在空气中红热时，观察其发出的光和热。接着，将红热的铁丝迅速放入纯氧气的环境中，观察其燃烧的剧烈程度，记录反应中的变化。注意在进行此实验时要保持安全距离，避免烫伤或火灾。

【加热高锰酸钾制取氧气】 : 首先，准备一个干燥的试管，加入适量的高锰酸钾，并将试管固定在试管夹中。然后，将试管放置在一个加热器上，缓慢加热高锰酸钾，观察反应过程中是否有气体产生。使用水槽收集气体，若有气泡生成，则说明在加热过程中产生了氧气。此过程需要持续加热，直到不再有气泡出现，最后安全地处理反应后的残留物。

【品红在水中扩散】 : 首先，准备一个透明的量杯，注入适量的清水。然后，取少量品红溶液，用吸管或滴管轻轻滴入水中，观察滴入后的颜色变化。记录品红溶液在水中的扩散过程，随着时间的推移，观察颜色的均匀程度及扩散范围的增长，注意观察扩散速度与水温等因素的关系。

【向浑浊的水中加入明矾粉末】 : 首先，取一杯浑浊的水，并准备适量的明矾粉末。然后，缓慢地将明矾粉末加入浑浊水中，轻轻搅拌使其充分溶解。观察混合后水的变化，记录明矾与水的反应过程以及水的澄清程度，注意反应过程中的沉淀情况。

【过滤】 : 首先，准备好一套过滤装置，包括漏斗和滤纸。将滤纸放入漏斗中，并将漏斗置于干净的收集容器上。然后，将加入明矾前的浑浊水缓慢倒入滤纸中，观察过滤后的水和滤纸上的沉淀物，记录过滤的效率和结果，分析水的净化效果。

【用肥皂水区分软水和硬水】 : 首先，准备两杯水，一杯为软水，一杯为硬水。然后，向每杯水中分别加入适量的肥皂水，并用搅拌棒轻轻搅拌。观察水中的泡沫情况，记录软水与硬水的泡沫产生情况和稳定性，分析肥皂在软水和硬水中的效果差异。

【实验室制取蒸馏水】 : 首先，准备好蒸馏装置，包括圆底烧瓶和冷凝管。将待蒸馏的水或混合溶液倒入圆底烧瓶中，并确保连接部分气密。然后，用加热器加热烧瓶中的液体，观察蒸气的产生并随其凝结成液体流入接收容器中，记录蒸馏水的收集量和质量。

【氢气在空气里的燃烧】 : 首先，准备一个装有氢气的试管，并确保试管内有适量的氢气。用木条或其他点火器具点燃一根引火条，然后迅速将其放入试管中。观察氢气在空气中的燃烧现象，并记录火焰颜色、声音（如“嘭”的一声）和温度变化，分析氢气燃烧的特点。

【盐酸与碳酸钠粉末反应前后质量的测定】 : 首先，称量适量的碳酸钠粉末并记录其质量，然后将其放入一个干燥的烧杯中。接着，逐渐往烧杯中加入适量的盐酸，观察反应现象（如气泡产生等），并在反应完成后再次称量烧杯中的剩余固体和液体，记录反应前后质量的变化，分析质量守恒定律。

【镁条燃烧前后质量的测定】 : 首先，准备一根镁条并称量记录其质量。然后，用点火器具点燃镁条，并将其放在一个干燥的烧杯中，观察镁条燃烧的现象（如光亮和烟雾）。燃烧结束后，等待冷却，再次称量烧杯中生成的白色粉末（氧化镁）和杯子的质量，记录并分析镁燃烧前后质量的变化。

【在加入红墨水的水里投入木炭】 : 首先，准备一个透明的容器，倒入适量的水并加入少量红墨水，搅拌均匀，使水呈红色。然后，取适量木炭，直接投入红墨水水中。观察木炭与水的反应过程，记录木炭对水颜色的影响及红墨水的扩散现象。

【用木炭还原氧化铜】 : 首先，准备一个干燥的坩埚，并在其内放入适量氧化铜。然后，在坩埚上方放置木炭，确保两者接触良好。使用酒精灯或其他加热设备对坩埚进行加热，观察反应过程中的颜色变化，并记录反应后生成的物质特征，分析木炭还原氧化铜的效果。

【倾倒二氧化碳】 : 首先，准备一个装有二氧化碳的气体瓶，并确保气密性良好。准备一个透明的容器，用来收集流出的二氧化碳气体。轻轻打开气体瓶的阀门，观察气体的释放情况，记录二氧化碳的气泡和流动特征，分析其在空气中的行为。

【二氧化碳与水的反应】 : 首先，准备一小瓶水和一瓶二氧化碳气体，确保气体瓶气密。然后，向水中缓慢注入二氧化碳，观察反应并记录溶液的变化，例如是否出现气泡和酸性变化。分析二氧化碳和水的反应生成碳酸的过程。

【燃烧条件的实验】 : 首先，准备几个已点燃的蜡烛和不同的条件（如有/无氧气的环境）。观察蜡烛在不同条件下的燃烧情况，并记录蜡烛的燃烧时间和火焰的特征，分析燃烧所需的条件（如氧气和温度）。

【粉尘爆炸实验】 : 首先，准备干燥的粉尘（如面粉或木屑），在安全的环境中进行实验。然后，在容器中放入适量的粉尘，并用点火器具点燃，观察其爆炸现象和强度，记录不同粉尘粒径和浓度对爆炸效果的影响，确保安全距离。

【观察氢氧化钙】 : 首先，准备一小瓶氢氧化钙溶液，倒入透明的容器中。观察溶液的颜色和透明度，并加入适量的水，以观察并记录溶解情况及反应现象，分析氢氧化钙的特性和用途。

【试验物质的导电性】 : 首先，准备几个不同的溶液（如盐水、糖水、蒸馏水），以及一套导电性测试装置（如电池、导线和灯泡）。将导线连接至电池的正负极，并分别将导线浸入各种溶液中，观察灯泡是否亮起，记录导电性的实验结果，分析不同溶液的导电性差异。

【在氢氧化钠溶液中滴加稀盐酸】 : 首先，准备一个装有氢氧化钠溶液的烧杯，使用滴管吸取适量的稀盐酸。慢慢地向氢氧化钠溶液中滴加盐酸，并观察溶液的变化，记录颜色变化和气体的产生，分析氢氧化钠与稀盐酸的中和反应。

【测定溶液的pH值】 : 首先，准备一系列待测的溶液样本，并采用pH试纸或pH计。依次将pH试纸浸入每种溶液中，观察颜色变化并与标准颜色图对比，记录pH值数据，分析不同溶液的酸碱性。

【测定生活中一些物质的pH】 : 首先，从日常生活中取出一些物质（如饮料、酸奶、肥皂水等），并准备pH试纸或pH计。逐个将试纸浸入各样本中，观察颜色变化并记录每种物质的pH值，分析其酸碱性及可能的化学性质。

【碳酸钠与盐酸反应】 : 首先，准备一个烧杯，加入适量的碳酸钠，并记录其质量。然后，缓慢地向烧杯中加入盐酸，观察并记录反应现象，如气泡的产生和温度变化，分析反应的方程式和性质，理解碳酸钠与盐酸的中和反应。

【向碳酸钠溶液里滴入澄清石灰水】 : 首先，准备一贝碳酸钠溶液和一小瓶澄清石灰水。用滴管将澄清石灰水缓慢滴入碳酸钠溶液中，观察反应现象，记录任何颜色变化或沉淀的生成，分析石灰水与碳酸钠溶液的反应结果。

【氢氧化钠溶液、氯化钡溶液与硫酸铜溶液的反应】 : 首先，准备三种溶液：氢氧化钠、氯化钡和硫酸铜。分别将氢氧化钠和氯化钡溶液缓慢倒入硫酸铜溶液中，观察并记录反应现象，特别是沉淀的形成情况，分析不同溶液之间的反应及其生成物。

【反复加热聚乙烯塑料】 : 首先，准备小块聚乙烯塑料放置在耐热容器中。使用加热器对聚乙烯塑料进行加热，观察其物理变化和化学反应，记录塑料在加热过程中狭聚与熔融的特性，分析聚乙烯塑料的热稳定性及降解特性。

【观察和描述——对蜡烛及其燃烧的探究】 : 首先，准备一根蜡烛和一个打火机。点燃蜡烛，观察火焰的形状与颜色，并记录蜡烛在燃烧时所产生的烟雾和气体。注意观察蜡烛燃烧的过程中，蜡烛是如何消耗氧气以及在什么情况下火焰熄灭，记录相关的现象和数据。

【对人体吸入的空气和呼出的气体的探究】 : 首先，准备一个干净的透明容器和一个气球。用气球吸入空气，然后快速将其转移到容器中，观察并记录气球内与容器内的气体状态。如有可能，可以通过燃烧小蜡烛来观察呼出气体与空气的差异。

【分解过氧化氢制氧气的反应中二氧化锰的作用】 : 首先，准备一个试管，加入适量的过氧化氢溶液。然后，加入几颗二氧化锰，观察反应的开始。记录气泡产生的情况，分析二氧化锰作为催化剂在该反应中所起的作用。

【分子运动现象】 : 首先，准备一些色素（如墨水）和一杯水。将色素滴入水中，观察色素在水中的扩散过程，记录其扩散速度和范围，分析分子的无规则运动现象。

【在元素周期表中查找元素】 : 准备一份元素周期表，选择几个特定的元素。记录每个元素的符号、原子序数和化学性质，进行小组讨论，分析各元素之间的关系。

【水的组成——电解水实验】 : 首先，准备一个电解槽，连接电源。将水倒入电解槽中并加入适量的电解质（如盐或硫酸），再将电极插入水中。接通电源，观察气体的生成和气泡的产生，分析水的分解过程和组成。

【反应前后物质的质量关系】 : 首先，准备一些化学物质，例如盐酸和碳酸钠。分别称量反应前的质量，然后将它们混合反应并记录反应后的质量，分析反应前后物质的质量关系，以验证质量守恒定律。

【实验室里制取二氧化碳的装置】 : 准备一个反应装置，将碳酸钙与盐酸混合。观察生成气体和反应的产物，记录气体的流出和反应的持续性，分析二氧化碳的制取过程。

【灭火的原理（下册）】 : 准备不同的灭火器材（如灭火器、水、沙子等），依次对燃烧的物体进行灭火实验。记录每种消防措施的效果，分析其原理并讨论相对有效性和安全性。

【金属与盐酸、稀硫酸的反应】 : 准备几种金属（如锌、镁）和盐酸、稀硫酸。将金属片放入酸溶液中，观察和记录反应现象，包括气体的产生和溶液的变化，分析不同金属与酸的反应特点。

【金属活动性顺序】 : 准备金属块样品及不同酸溶液。将金属样品依次与酸反应，观察其反应的剧烈程度。记录数据，比较和分析金属的活动性顺序。

【铁制品锈蚀的条件】 : 准备铁钉并将其放在不同的环境（如干燥的空气、水中、盐水中等）。观察和记录铁的锈蚀情况，讨论不同环境对铁锈蚀的影响。

【溶解时的吸热或放热现象】 : 准备一杯水，分别加入不同的溶质（如氯化钠、氯化铵）。观察在溶解过程中的温度变化，并记录数据，以分析溶解时的吸热或放热现象。

【溶解度曲线】 : 准备不同温度下的溶质溶解度实验, 记录每种溶质在不同温度下的溶解量, 绘制溶解度曲线，分析溶解度与温度的关系。

【自制酸碱指示剂】 : 准备一些紫甘蓝，进行切碎并煮沸提取指示剂。将提取到的指示剂与不同的酸和碱进行反应，观察颜色变化，记录值并分析结果。

【酸的化学性质】 : 准备不同的酸（如盐酸、醋酸），分别与不同反应物（如碳酸钠、金属等）进行反应，观察并记录反应结果和现象，分析酸的化学性质。

【碱的化学性质】 : 准备不同的碱（如氢氧化钠、氢氧化钙），与不同物质（如酸、金属等）进行反应，观察并记录反应现象，分析碱的化学性质。

【某些酸、碱、盐之间是否发生反应】 : 准备不同的酸、碱和盐样品，依次将它们混合观察反应现象，记录数据，探索各种化学反应之间的关系。

【初步区分常用氮肥、磷肥和钾肥的方法】 : 准备不同类型的肥料样本，通过溶解和反应的方法分别检测氮、磷、钾的特征，以初步区分常用的氮肥、磷肥和钾肥。

【氧气的实验室制取与性质】 : 首先，准备一个装有过氧化氢溶液的烧瓶，并加一小撮二氧化锰作为催化剂。用热源加热烧瓶，使其产生氧气。观察生成的氧气并收集到气体收集瓶中。可以用带火星的木条测试气体的性质，观察木条是否复燃，记录结果。

【二氧化碳的实验室制取与性质】 : 首先，准备一个装有碳酸钙的试管，并向其中缓慢滴加稀盐酸。观察气体的产生并记录反应现象（如气泡生成）。收集气体，并用清水测试其是否使石灰水变浑浊，记录这一性质的表现。

【燃烧的条件】 : 准备一个蜡烛和一个瓶子。点燃蜡烛后，将瓶子反转覆盖在蜡烛上。观察蜡烛燃烧，并注意在瓶子内气体的变化。记录蜡烛在缺氧环境下的燃烧时间及现象，分析燃烧所需的条件（如氧气的供给）。

【金属的物理性质和某些化学性质】 : 准备几种金属样品（如铜、铝、铁等），观察它们的颜色、硬度、熔点等物理性质。然后，可以尝试将金属与酸反应，记录反应速率、生成气体的情况等，分析金属的化学性质。

【一定溶质质量分数的氯化钠溶液的配制】 : 首先，选择一特定质量的氯化钠，用天平准确称取所需质量，记录数据。然后，准备一定体积的蒸馏水，将氯化钠加入水中，用玻璃棒搅拌均匀，直至全部溶解。记录溶液的颜色和透明度，并分析溶液的质量分数。

【粗盐中难溶性杂质的去除】 : 在实验中，准备适量的粗盐，加入适量的水使其溶解。然后，使用过滤器过滤混合物，去除不溶性杂质，收集滤液并记录观察的颜色和透明度，分析过滤后的盐水质量变化。
